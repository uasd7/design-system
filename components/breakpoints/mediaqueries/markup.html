Mixins für MediaQueries


from

Breakpoints kleiner als value werden nicht berücksichtigt

@include media-breakpoint-from(value) { ... }

value: m = xxxs, xxs, xs, s, m, l, xl

to

Breakpoints vom kleinsten bis einschließlich value werden angesprochen

@include media-breakpoint-to(value) { ... }

value: m = xxxs, xxs, xs, s, m , l, xl


between

Breakpoints von value from bis einschließlich value to werden angesprochen

@include media-breakpoint-between(value from, value to) { ... }

values: s, l = xxxs, xxs, xs, s, m, l, xl

only

Ausschließlich der Breakpoint von value wird angesprochen

@include media-breakpoint-only(value) { ... }

value: m = xxxs, xxs, xs, s, m, l, xl
